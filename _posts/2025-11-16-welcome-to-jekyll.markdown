תמצאו את הפוסט הזה בתיקייה `_posts`. המשיכו קדימה וערכו אותו ובנו מחדש את האתר כדי לראות את השינויים שלכם. ניתן לבנות מחדש את האתר בדרכים רבות ושונות, אך הדרך הנפוצה ביותר היא להריץ `jekyll serve`, אשר מפעיל שרת אינטרנט ומייצר מחדש באופן אוטומטי את האתר שלכם כאשר קובץ מתעדכן.You’ll find this post in your `_posts` directory. Go ahead and edit it and re-build the site to see your changes. You can rebuild the site in many different ways, but the most common way is to run `jekyll serve`, which launches a web server and auto-regenerates your site when a file is updated.



Jekyll דורש שקבצי פוסטים בבלוג יקראו בהתאם לפורמט הבא:Jekyll requires blog post files to be named according to the following format:



`YEAR-MONTH-DAY-title.MARKUP``YEAR-MONTH-DAY-title.MARKUP`



כאשר `YEAR` הוא מספר בן ארבע ספרות, `MONTH` ו-`DAY` הם שניהם מספרים בני שתי ספרות, ו-`MARKUP` היא סיומת הקובץ המייצגת את הפורמט המשמש בקובץ. לאחר מכן, כללו את ה-front matter הנדרש. העיפו מבט בקוד המקור של פוסט זה כדי לקבל רעיון על איך זה עובד.Where `YEAR` is a four-digit number, `MONTH` and `DAY` are both two-digit numbers, and `MARKUP` is the file extension representing the format used in the file. After that, include the necessary front matter. Take a look at the source for this post to get an idea about how it works.

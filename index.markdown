# קורס קובול בעברית

ברוכים הבאים לקורס קובול המקיף בעברית! כאן תלמדו את כל מה שצריך לדעת כדי להתחיל לתכנת בקובול, שפת התכנות הוותיקה שעדיין מניעה את העולם הפיננסי.

## 📚 מה תלמדו בקורס

בקורס נכסה את הנושאים הבאים:
* מבוא לקובול וההיסטוריה של השפה
* התקנת סביבת עבודה וכלים נדרשים
* יסודות התכנות בקובול: משתנים, מבני נתונים, ובקרת זרימה
* עבודה עם קבצים ועיבוד נתונים
* תכנות מתקדם: תוכניות משנה, טבלאות ועוד
* פרויקט גמר: בניית מערכת בנקאית פשוטה
